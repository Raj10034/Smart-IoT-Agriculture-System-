# 📌 Project Overview

The Smart IoT Agriculture System is an IoT-based solution designed to improve irrigation efficiency and support smart farming. The system monitors soil moisture, temperature, humidity, and water tank levels using sensors connected to a NodeMCU ESP8266 microcontroller.

The system enables both **automatic and manual control of irrigation pumps** through a mobile application. Farmers can monitor real-time environmental conditions and manage irrigation remotely using a smartphone.

This project helps reduce water wastage, improve crop productivity, and provide sustainable irrigation management using modern IoT technology.

# ❗ Problem Statement

Traditional irrigation systems require manual monitoring and control, which can lead to inefficient water usage and increased labor. Farmers often lack real-time information about soil conditions and water availability.

This project addresses these challenges by implementing an **IoT-based automated irrigation system** that monitors environmental conditions and controls water pumps efficiently.

---

# 💡 Proposed Solution

The system uses sensors connected to a **NodeMCU ESP8266** microcontroller to monitor soil moisture, temperature, humidity, and water tank levels.

The system:

- Automatically activates irrigation when soil moisture is low
- Prevents pump operation when water tank levels are insufficient
- Allows manual control through a mobile application
- Displays real-time data on an LCD and mobile dashboard

---

# ⚙️ System Architecture
Farmer (Mobile App) → Internet / WiFi → NodeMCU ESP8266 → Sensors (Soil Moisture, Temperature, Humidity) → Relay Module → Water Pump / Irrigation System

# 🏗 Project Workflow

1. Sensors collect environmental data (soil moisture, temperature, humidity).
2. NodeMCU processes the sensor data.
3. If soil moisture is low, the system activates the irrigation pump.
4. Water tank levels are checked before activating the pump.
5. Data is sent to the mobile application for remote monitoring.
6. Farmers can manually control irrigation through the app.
